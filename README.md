# Toronto AirBnB Listings Analysis

This repo contains a quick analysis of the Toronto AirBnB listings.

### Requirements

* Python (3.5+)
* Pandas
* Matplotlib
* Numpy
* Scikit-learn
* XGBoost (optional)


### Getting Started

You can run the Jupyter (iPython) notebook with the following command:

```
jupyter notebook ./airbnb.ipynb
```
